# ROAD FIGHTER X86

A clone to the classic game developed by KONAMI for NES

## INSTRUCTIONS

Use DOS-BOX & NASM to run the following code: 
1. nasm RoadFighter.asm -o RoadFighter.com
2. RoadFighter.com
